# Security Policy

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report vulnerabilities through [GitHub private vulnerability reporting](https://github.com/freeCodeCamp-Universe/universe-cli/security/advisories/new). A maintainer will acknowledge receipt within 5 business days and coordinate a fix and disclosure timeline with you.

## Supported Versions

Only the latest minor release line receives security fixes. See [Releases](https://github.com/freeCodeCamp-Universe/universe-cli/releases) for the current version.

## Scope

This policy covers the `@freecodecamp/universe-cli` package and the published release binaries. Vulnerabilities in upstream dependencies should be reported to the respective upstream projects; you may also file a private advisory here if the issue materially affects this CLI.
